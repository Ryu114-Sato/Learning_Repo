# Markdown ファイルの書き方

Markdown は、軽量でシンプルな記法を使用してテキストを整形するためのマークアップ言語です。以下に基本的な Markdown の構文を示します。

## 見出し

見出しは`#`記号を使用して作成します。  
`#`の数で見出しのレベルを指定します。

```markdown
# 見出し 1

## 見出し 2

### 見出し 3
```

# 強調
